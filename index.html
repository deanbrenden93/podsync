🎤 Microphone blocked

Your browser remembers that you clicked "Block" for this site. We can't re-ask from inside the app — you need to un-block it manually.
Chrome / Edge
1. Click the 🔒 lock icon to the left of the URL.
2. Find Microphone in the dropdown.
3. Set it to Allow.
4. Reload the page (F5).
Firefox
1. Click the 🔒 lock icon to the left of the URL.
2. Click the > next to "Permissions".
3. Remove the block on "Use the Microphone".
4. Reload the page (F5).
Safari
1. Safari menu → Settings → Websites → Microphone.
2. Find this site and set it to Allow.
3. Reload the page (⌘R).
